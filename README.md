# der-pflegeberater.de

**Onepager – Vergütungsverhandlungen ambulante Pflege**

by Roberto Bonavita (opensource-data)

## Überblick

Dieser Onepager richtet sich an ambulante Pflegedienste, Träger und Leitungen in Nordrhein-Westfalen und darüber hinaus, die ihre Vergütungsverhandlungen nach SGB XI strukturiert vorbereiten möchten.

## Technologie

- Reines HTML/CSS/JavaScript, keine externen Abhängigkeiten
- Drag-and-drop Dateivorbereitung lokal im Browser
- Kontaktweiterleitung via Telegram, ToffeeShare oder E-Mail
- Responsives Layout (Mobile, Tablet, Desktop)

## Bilder

Die Bilddateien werden lokal referenziert. Bitte in den Ordner `img/` legen:

- `img/portrait.jpg` – Porträt
- `img/scene.jpg` – Illustrationsszene
- `img/consulting.jpg` – Beratungsszene
